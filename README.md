## Getting started

Soda Core currently supports PosgreSQL, Redshift, BigQuery, and Snowflake. You can get started by installing appropriate package using `pip`:

* `soda-core-postgres`
* `soda-core-redshift`
* `soda-core-bigquery`
* `soda-core-snowflake`

e.g. `pip install soda-core-postgres`
